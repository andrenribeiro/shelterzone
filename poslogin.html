MAN IN THE MIDDLE

Um dos ataques mais difundidos na internet, o man in the middle consiste no desvio do fluxo de dados de uma máquina para outra por meio de um intermediário. Dessa forma, o atacante pode filtrar e obter dados da vítima, como a senha de um endereço de e-mail.

COMO OCORRE?

O hacker atua alterando o endereço MAC para se passar pelo endereço IP da maquina vítima

COMO SE PROTEGER?

Nesta atividade, o aluno deve inserir o endereço de IP estaticamente de forma que toda vez que a máquina for reiniciada, o IP estará associado ao endereço MAC correto. Para isso, deve-se abrir o prompt de comando elevado - que requer autorização do sistema - e inserir os dados a seguir, nos quais 192.XXX.X.X é o seu endereço de IP e c8.5d.xxxxxx o MAC address associado

SQL INJECTION

Como funciona esse tipo de ataque?

A injeção de SQL é um ataque cibernético que engana um banco de dados para permitir o acesso de hackers. Uma injeção de SQL força que um banco de dados desprotegido execute comandos inseguros ao inserir código maligno no SQL do banco de dados, a linguagem mais usada para gerenciamento de banco de dados.

O exemplo do nosso simulador consiste em uma página de login de um banco na qual um hacker pode aplicar esse golpe. Como o aluno deve então se defender?
